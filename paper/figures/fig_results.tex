\caption{Why duplicate-aware fusion wins, on the independent 28-scene reference.
\textbf{(a)} Object-label F1 vs.\ number of input views: every prior cross-view fusion variant
\emph{degrades} as views are added (more frames yield more un-collapsed duplicates), while
duplicate-aware fusion (amber) is the only variant whose accuracy \emph{rises} and is best at every
view count; the uncertainty gate (brick) sits below even the uninformed control.
\textbf{(b)} The mechanism: dedup \emph{recovers precision} (solid amber far above the baseline,
$0.71\!\to\!0.59$ vs.\ $0.56\!\to\!0.40$) while leaving \emph{recall essentially unchanged} (dashed
lines overlap). Values match Table~\ref{tab:variant_f1}.}
